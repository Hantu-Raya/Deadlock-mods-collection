# Deadlock Mods Collection

A personal collection of custom HUD and gameplay modifications for **Deadlock**.  
These scripts are designed to enhance the in-game UI, providing better visibility for health, buffs, shopping history, and more.

## Included Mods

### HUD & Interface
- **buff_timer_top_bar**: Displays active buff timers directly on the top bar.
- **hud**: General HUD enhancements and tweaks.
- **radar**: Modifications to the radar/minimap visibility.
- **showrank**: Displays rank information in-game.
- **unlockprogress**: Tracks and displays unlock progression.
- **legacytarget**: Restores or modifies the legacy targeting interface.

### Health & Status
- **hp / self_hp**: Customizations for health bar visualization (self and enemies).
- **heatlh_color_blind**: Accessibility improvements for health bars.
- **soma passive**: Visual indicators for Soma's passive abilities.

### Utility
- **recent_purchase**: Tracks and displays recent item purchases.
- **kaiz special**: Custom configuration script.

## Setup
Most of these mods are likely intended to be used with the Deadlock scripting/UI system.
1. Copy the desired mod folders to your game's modification directory.
2. Ensure any dependencies are installed.

## Development
- **Source**: The `src` or root folders contain the raw scripts.
- **Compiled**: The `*_compiled` folders are generated builds (excluded from this repo).
